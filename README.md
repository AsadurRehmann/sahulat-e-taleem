# Sahulat-e-Taleem
**Centralized Scholarship Matching Portal for Pakistani Students**

Sahulat-e-Taleem is a full-stack web application designed to bridge the information gap for students in Pakistan. The platform aggregates National (HEC, PEEF, BEEF), International, and Need-based scholarships, matching them to users based on academic and financial profiles.

## Key Features
- **Smart Matching Engine**: Filters scholarships based on GPA, Province, and Family Income.
- **Student Dashboard**: Personalized recommendations based on real-time profile data.
- **Profile Management**: Interface for users to update educational and financial details.
- **Administrative Portal**: Secure gateway for manual scholarship verification and data entry.
- **Responsive UI**: Optimized for mobile and desktop accessibility.

## Tech Stack
- **Frontend**: React.js (Vite), TypeScript, Tailwind CSS
- **UI Components**: shadcn/ui, Lucide React
- **Backend/Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth (JWT)
